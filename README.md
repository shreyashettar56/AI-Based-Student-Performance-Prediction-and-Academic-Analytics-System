# AI-Based-Student-Performance-Prediction-and-Academic-Analytics-System

📌 Project Overview
The AI-Based Student Performance Prediction and Academic Analytics System is a machine learning-based web application developed to predict student academic performance using various educational factors. The system analyzes inputs such as study hours, attendance, and previous scores to estimate student exam performance and provide academic insights.

The project combines Machine Learning, Data Analytics, and Web Application Development to create an interactive and user-friendly academic prediction platform.

🎯 Objectives
* Predict student academic performance using Machine Learning
* Analyze relationships between study habits and exam scores
* Provide intelligent academic recommendations
* Maintain student prediction records
* Visualize student performance data using graphs and analytics

📊 Dataset
The dataset used for this project was obtained from Kaggle and contains student-related academic factors such as:
* Hours Studied
* Attendance
* Previous Scores
* Motivation Level
* Sleep Hours
* Exam Score

The system uses selected features to train machine learning models and generate predictions.

⚙️ Working  the System
1. User logs into the application
2. Student details are entered
3. Machine Learning model processes the input data
4. Predicted exam score is generated
5. Grade and performance level are displayed
6. AI-based academic recommendations are provided
7. Student records are stored in the database
8. Analytics and visualizations are generated
